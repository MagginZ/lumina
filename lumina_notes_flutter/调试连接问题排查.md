# Dart VM Service 连接失败 - 排查步骤

## 方法一：用 Xcode 运行并查看日志（推荐）

1. 打开项目：
```bash
cd lumina_notes_flutter
open ios/Runner.xcworkspace
```

2. 在 Xcode 中：
   - 顶部选择你的 iPhone 作为目标设备
   - 菜单 **Product → Run**（或按 ⌘R）

3. 观察：
   - 若应用能正常打开，说明应用本身没问题，只是 `flutter run` 的调试连接异常
   - 若闪退，在 Xcode 底部 **Console** 中查看崩溃原因（红色错误信息）

---

## 方法二：直接从手机主屏幕启动

若应用已安装到手机：

1. 在 iPhone 主屏幕找到「Lumina 笔记」
2. 直接点击打开

若能正常使用，说明应用没问题，可以暂时不用 `flutter run` 调试。

---

## 方法三：确认以 Debug 模式运行

```bash
cd lumina_notes_flutter
flutter run --debug
```

避免使用 `--release` 或 `--profile`，这两种模式没有 Dart VM 调试服务。

---

## 方法四：查看详细日志

```bash
flutter run -v
```

根据输出判断是应用未启动、崩溃，还是网络/连接问题。
